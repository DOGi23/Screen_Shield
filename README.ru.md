# ScreenShield (Бесплатная Open-Source Версия)

ScreenShield — это легкая и высокопроизводительная утилита для Windows, разработанная для предотвращения захвата элементов рабочего стола, ярлыков, обоев и окон конкретных приложений программами записи экрана, трансляции и стриминга (такими как OBS, Discord, Zoom и др.).

---

## 🌟 Ключевые возможности

- **🔒 Исключение окон в реальном времени**: Скрывает выбранные запущенные программы от инструментов захвата экрана с помощью системного API (Display Affinity), оставляя их полностью видимыми для вас.
- **🖼️ Защита рабочего стола и обоев**: Автоматически скрывает обои и ярлыки рабочего стола во время захвата стриминговыми программами.
- **⌨️ Глобальные горячие клавиши**: Быстрое включение и выключение защиты на лету с помощью комбинации `Ctrl + Shift + S`.
- **🔊 Звуковой отклик (Audio Feedback)**: Мгновенные звуковые сигналы при переключении режима защиты, позволяющие контролировать статус защиты прямо в играх без необходимости сворачивать окно.
- **🔔 Уведомления (Toasts)**: Всплывающие окна для индикации статуса защиты.
- **📂 Drag & Drop обоев**: Возможность быстро установить защитные обои, просто перетащив любое изображение (PNG, JPG и др.) прямо в интерфейс программы.
- **🚀 Автозапуск**: Опция запуска приложения при старте Windows в свернутом виде (в системный трей).
- **🔗 Скрытие из панели задач (Опционально)**: Динамическое скрытие кнопок защищаемых программ из нижнего бара (панели задач Windows) с использованием COM-интерфейсов для максимальной приватности.

---

## 🛠️ Технологический стек

- **Платформа**: .NET 8.0 / WPF (Windows Presentation Foundation)
- **Язык**: C# 12
- **Системные вызовы**: Windows Native APIs (User32, Shell32 через P/Invoke, COM-интерфейсы `ITaskbarList`)
- **Запросы системы**: `System.Management` для WMI-запросов

---

## 🚀 Сборка и Запуск

### Требования
- Установленный [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) или новее.
- Visual Studio 2022 (с установленной рабочей нагрузкой «Разработка классических приложений .NET») или VS Code.

### Сборка через интерфейс командной строки (CLI)
Откройте консоль/PowerShell в корневой папке проекта и выполните команду:
```bash
# Восстановление зависимостей и сборка в конфигурации Release
dotnet build -c Release
```

Скомпилированные файлы приложения будут находиться по пути:
`bin/Release/net8.0-windows/ScreenShield.exe`

## 📦 Структура проекта

- `MainWindow.xaml` / `MainWindow.xaml.cs` — Главное окно приложения, привязка настроек и обработка событий интерфейса.
- `DesktopManager.cs` — Управление оверлеем для защиты ярлыков и подменой обоев.
- `ProcessShieldManager.cs` — Мониторинг процессов Windows для их автоматического скрытия от захвата.
- `AppConfig.cs` — Логика загрузки и сохранения параметров конфигурации в JSON.
- `AntiDebug.cs` — Инструмент проверки для обнаружения отладчиков.
- `NativeMethods.cs` — Импорт нативных Win32 функций API.
